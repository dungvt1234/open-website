OpenClaw

Bản HTML đọc nhanh của bộ slide 25 trang, có kèm speaker notes.

Slide 2. OpenClaw là gì?

  • Nền tảng AI agent có thể trò chuyện, hành động, và tự động hóa
  • Hỗ trợ nhiều kênh: Telegram, WhatsApp, Discord, Signal...
  • Có thể dùng tools để đọc file, sửa file, chạy lệnh, tìm web
  • Hỗ trợ automation, multi-agent, skills, plugins, ACP

Slide 3. OpenClaw dùng để làm gì?

  • Trợ lý cá nhân
  • Coding assistant
  • Bot vận hành nội bộ
  • Agent đa kênh cho nhóm
  • Hệ thống automation theo lịch hoặc theo sự kiện

Slide 4. Kiến trúc tổng quan

  • Gateway: trung tâm điều phối
  • Agent: AI xử lý hội thoại và hành động
  • Workspace: file, memory, skill, ghi chú
  • Channels: nơi nhận và gửi tin nhắn
  • Tools: năng lực thao tác hệ thống
  • Plugins/Skills: lớp mở rộng và chuyên biệt hóa

Slide 6. Cài đặt nhanh

curl -fsSL https://openclaw.ai/install.sh | bash
openclaw onboard --install-daemon

Kiểm tra:

openclaw gateway status
openclaw dashboard

Slide 7. Onboarding làm gì?

  • Chọn model provider
  • Cấu hình API key hoặc auth
  • Thiết lập gateway
  • Thiết lập daemon/service
  • Chuẩn bị workspace ban đầu

Slide 8. File cấu hình quan trọng nhất

~/.openclaw/openclaw.json
  • agents
  • channels
  • gateway
  • tools
  • skills
  • cron / hooks / automation

Slide 9. Ví dụ config tối thiểu

{
  agents: { defaults: { workspace: "~/.openclaw/workspace" } },
  channels: {
    telegram: {
      enabled: true,
      botToken: "123:abc",
      dmPolicy: "pairing"
    }
  }
}

Slide 11. Agent, session, workspace

  • Agent: một AI persona / runtime cụ thể
  • Session: giữ ngữ cảnh hội thoại
  • Workspace: nơi agent đọc, ghi, nhớ, và làm việc

Slide 12. Memory và file ngữ cảnh

  • SOUL.md
  • USER.md
  • MEMORY.md
  • memory/YYYY-MM-DD.md
  • AGENTS.md

Slide 13. Tools, sức mạnh thật sự của agent

  • File: read, write, edit
  • Runtime: exec, process
  • Web: web_search, web_fetch
  • Automation: cron
  • Session: sessions_*, subagents
  • Media: image, tts, video_generate

Slide 14. Skills và plugins

  • Skills: dạy agent khi nào dùng tool và dùng ra sao
  • Plugins: thêm channel, provider, tool, media, runtime mới
openclaw skills list
openclaw plugins list

Slide 15. Kết nối channel, ví dụ Telegram

{
  channels: {
    telegram: {
      enabled: true,
      botToken: "123:abc",
      dmPolicy: "pairing",
      groups: { "*": { requireMention: true } }
    }
  }
}

Slide 18. Heartbeat, nhịp kiểm tra định kỳ

{
  agents: {
    defaults: {
      heartbeat: { every: "30m", target: "last" }
    }
  }
}

Slide 19. Cron, tự động hóa đúng giờ

openclaw cron add \
  --name "Morning brief" \
  --cron "0 7 * * *" \
  --session isolated \
  --message "Summarize overnight updates." \
  --announce

Slide 20. Cron vs Heartbeat

  • Cần đúng giờ → cron
  • Cần kiểm tra định kỳ có ngữ cảnh → heartbeat

Slide 21. Multi-agent, khi một agent không còn đủ

  • main: trợ lý chung
  • coder: xử lý code
  • ops: monitoring và automation
  • support: trả lời khách hàng
